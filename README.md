## Unreal Editor 화면
![](github_resources/Editor.png)
- ViewPort
  - 현재 작업중인 맵이 보여지는 화면
  - 액터를 배치하고 이동, 회전, 스케일링 하는 등의 작업이 가능함
- Docked Layout
  - Content Browser(Drawer), OutputLog 등의 창을 고정 가능한 레이아웃
- Outliner
  - 뷰포트에 배치된 액터들을 확인할 수 있음
  - 폴더를 생성해 묶어서 관리할 수 있음
- Details
  - 뷰포트의 액터를 선택해 위치, 회전 등의 작업과 컴포넌트 추가 등을 지원해주는 창
